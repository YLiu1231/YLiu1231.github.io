用来摸鱼的座位就是要舒服一点

发表于 2021-09-10 更新于 2021-09-11 分类于 Life 阅读次数：

0. 展示

1. 桌面配置介绍

  • 电脑: 一台15-inch mid 2015 MBP; 一台同组小伙伴帮组装的垃圾桶

    • Mac已经用了三年了，是去17年美国交换之前老爸送的礼物（一直到现在都是主力机）
    • pc的配置:
      • 处理器: Intel Core i5-8600K @ 3.60GHz
      • 主板: MSI Z370M MORTAR (MS-7B54)
      • 内存: 16 GB ( CORSAIR DDR4 3000MHz )
      • 主硬盘: Intel SSDPEKKW256G8 ( 256 GB ssd )
      • 显卡: Nvidia GeForce GTX 1070 ( 8 GB / ZOTAC )
      • 显示器: DELL U2518D / DELL P2415Q

  • 设备: Magic TrackPad2/ Varmilo VA87MAC（有线)/ 罗技G502

    ​ Varmilo VDBox/ 华为无线充电板15W/ 乂度TA-10/ AKG-k812/ QDC8CH/ AKG-N5005

    ​ 希捷移动硬盘2TB、Samsung移动硬盘2TB/ USB3.0一拖四拓展坞

  • 支架: 爱格升45-295-026

  • 壁纸: Clannad动感新势力赠送海报

    ​ 香港机场平面图（一年级做了一个机场货运项目）

  • 零件: 文具都是MUJI买的，特别好用

2. 连接方式

显示器

  • 2\(\times\)MAC Thunderbolt2 (miniDP) 连接两台显示器(DP)

  • 2\(\times\)显卡DP口连接两台显示器(miniDP)

这样2乘2连接的方式可以让我两台设备都有两个显示器可以用，也可以任意一块显示器显示任意设备，非常自由。缺点是需要四根较长miniDP转DP线材，才能顺利连接并且将线材隐藏到支架内

数据传输

  • MAC USB-A连接P2415Q USB-B (3.0)

  • PC USB-A连接U2518D USB-B (3.0)

打通了数据通道后，显示器本身可作为拓展坞使用（不要太方便！），并且不需要另外给拓展坞供电。现在挂在显示器后面的一拖四拓展坞是之前购买的，需要独立供电，导致线材十分凌乱

音源输出

  • 乂度TA-10 AUX1口连接MAC 3.5mm耳机孔
  • 乂度TA-10 USB-B连接pc USB-A

由于TA-10是解码耳放一体机，支持多种音源输入方式，并且按下旋钮即可切换，十分方便，让我可以在两台设备间自由切换，k812连TA-10小烧怡情

QDC8CH作为便携设备，我的手机HUAWEI Mate30Pro/iPhoneX可以直推无压力

键盘/鼠标/触摸板输入

  • 键盘通过VDBox连接U2518D，有线直连PC
  • 键盘通过VDBox蓝牙通道1连接MAC

这款键盘是给在阿米洛淘宝旗舰店看到的，十分喜欢。然而一个键盘没办法在两个设备间切换，就又买了他家的小玩意VDBox。本身由显示器供电，作为蓝牙发射器与MAC连接。剩下两个通道暂时空着（我也试过连手机，但是很少用到）。VDBox通过按下对应按键来切换连接对象，也算是一键切换吧（就是要伸下手。。。）

  • 鼠标连接U2518D，有线直连PC
  • 触摸板通过蓝牙连接MAC

其实我也想换个蓝牙键盘（Filco之类的一键切换连接对象的那种），但是蓝牙设备过多可能会导致不稳定（或许吧）（（更重要的是没钱。。。。））

移动硬盘

两个移动硬盘都是通过显示器后面的USB-A接口连接两台设备，不用担心供电问题

无线充电板

无线充电板连接的是左边的P2415Q，这样右侧的桌面就全无一物，平时看paper、上网课记笔记等需要很多空间的时候就在这边做

4.其他角度展示

PS

  1. 两个显示器是跟老板做了两个项目得的，感恩~

  2. 花了将近三个星期搞了这套设备（所有设备都是读博两年半攒下来的，主要是设计使用方式、走线，购买器材与等待时间），尽管依然有很多改进的空间，心里还是十分开心的。

  3. 我这次是换了座位才下决心重新整理桌面的，为了组内其他项目成员（我的亲师弟们）沟通交流方便，我从之前的宝地换到了办公室pantry的前面，打印机、冰箱、饮水机就在我的背后，全办公室人走来走去的时候都会瞄我几眼，那感觉。。。像景区。。。所以，就算摸鱼，我也要显得很认真（逛B站的我最专心，毕竟是抱着学习的心态欣赏up主们的才华的）

    我是一个有良知的人，我将无愧于父母的养育之恩和老师的良苦用心！

    我是一个有骨气的人，我不会甘于平庸、碌碌无为，我将通过百倍努力去改变人生、创造辉煌！

    我是一个意志坚强的人，我决不会言苦言累，“头悬梁，锥刺股”将是我学习上永远的榜样！
